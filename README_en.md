## Cognitive Types Hypothesis (Working Title)

## Overview
This repository manages draft papers related to the Cognitive Types Hypothesis.
The root folder contains numerous working notes and discussion files, but these are excluded from Git management.

## Directory Structure
- `paper/` : Directory for draft and final versions of the paper
- `paper/draft.md` : Main body of the paper
- `paper/notes_index.md` : Reference list for miscellaneous notes needed for the paper

## Tools Used
- Markdown, VSCode, Git

## Operational Rules
- Commit draft paper updates to `paper/`
- Edit miscellaneous notes freely locally
- Only `paper/` is managed on GitHub
