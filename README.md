[Click here to read the English README](README_en.md)

## 概要
このリポジトリは、Cognitive Types Hypothesis に関する論文草案を管理するためのものです。
root フォルダには作業用のメモや考察ファイルが多数存在しますが、Git 管理の対象外です。

## ディレクトリ構成
- `paper/` : 論文草案・完成版を置くディレクトリ
- `paper/draft.md` : メインの論文本文
- `paper/notes_index.md` : 論文に必要な雑多メモの参照リスト

## 使用ツール
- Markdown, VSCode, Git

## 運用ルール
- 論文草案の更新は `paper/` にコミット
- 雑多メモはローカルで自由に編集
- GitHub には `paper/` のみ管理
